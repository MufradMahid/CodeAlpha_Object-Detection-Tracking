# CodeAlpha_Object-Detection-Tracking
This project implements real-time object detection and tracking using YOLOv8 and Deep SORT. Objects are detected from a video file, assigned unique IDs and tracked across frames. Built with Python and OpenCV to demonstrate practical computer vision and deep learning skills.

## Description
This project performs real-time object detection and tracking using YOLOv8 and Deep SORT.
Detected objects are assigned unique IDs and tracked across video frames.

## Technologies Used
- Python
- OpenCV
- YOLOv8
- Deep SORT

## Features
- Real-time object detection
- Unique object ID tracking
- Webcam and video file support

## How to Run
1. Install dependencies:
   pip install ultralytics opencv-python deep-sort-realtime
2. Run:
   python main.py
